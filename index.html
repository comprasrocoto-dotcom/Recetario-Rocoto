Integración con GitHub

🐙

Este recetario vive en GitHub Pages

Todas las fotos, recetas personalizadas y notas se almacenan en tu repositorio. Las cocinas pueden acceder desde cualquier dispositivo con conexión.

📁 Estructura del Repositorio
malanga-recetario/
├── index.html          ← Esta app
├── recipes_data.json   ← Base de datos de recetas
├── photos/             ← Fotos de cada plato
│   ├── bocados-del-oriente.jpg
│   ├── pesca-malanga.jpg
│   └── ...
├── notes/              ← Notas por receta (JSON)
└── custom_recipes.json ← Recetas nuevas agregadas
🚀 Cómo publicar en GitHub Pages
  1. Crea un repositorio en github.com (ej: malanga-recetario)
  2. Sube los archivos: index.html + recipes_data.json
  3. Ve a Settings → Pages y activa GitHub Pages
  4. La URL será: https://tu-usuario.github.io/malanga-recetario
  5. Agrega fotos a la carpeta photos/ con el nombre del plato
  6. Las cocinas acceden desde celular, tablet o PC — sin instalar nada
📸 Cómo agregar fotos

Abre la receta → toca "Agregar Foto" → sube la imagen.
Para que persista entre sesiones, sube el archivo a la carpeta photos/ de tu repo con el nombre:

NOMBRE-DEL-PLATO.jpg (en minúsculas, espacios como guiones)

Agregar Foto

Sube una foto del plato. Se guardará localmente en tu dispositivo.

📷

Toca para seleccionar una foto

JPG, PNG, WEBP — máx 5MB
